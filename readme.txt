### M3D_DLL v1.01 hotfix release ###

1. Navigate to the Assets/Scripts/Editor folder in the MCS figure files, delete the contents.
2. Copy the contents of the the hotfix Editor folder into the project Assets/Scripts/Editor folder.
3. Navigate to the Assets/Plugins folder, delete the M3D_DLL file.
4. Copy the M3D_DLL_v1.01.dll file from the hotfix Plugins folder into the project Assets/Plugins.
5. Reimport the project Assets/MORPH3D folder.
